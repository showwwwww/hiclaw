# hiclaw

A cross-platform desktop UI for openclaw with voice command support. Features a modular character model system that accepts industry-standard formats, allowing users to customize and replace models with their own creations.

## Platform Support

- ✅ Windows 10/11
- ✅ macOS 10.15+

## Features

- Desktop interface for openclaw
- Independent voice-to-command control
- Modular model system - swap the default model with your own
- Supports common game industry model formats (FBX, glTF, OBJ, etc.)
- DIY-friendly architecture for community customization

## Use Cases

- Control openclaw hands-free with voice commands
- Personalize your experience with custom character models
- Integrate your own 3D assets using standard formats
